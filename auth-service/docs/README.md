## Ключевые особенности
- Регистрация и вход пользователей с поддержкой JWT и OAuth2.
- Управление профилями пользователей и настройками.
- Интеграция с внешними провайдерами аутентификации (Google, Facebook). (в планах)
- Безопасное хранение паролей с использованием bcrypt.
- Поддержка восстановления пароля и подтверждения по email.
- Логирование и мониторинг активности пользователей.

---
## Быстрый старт
1. `gradle :auth-service:bootRun`
2. Сервис будет доступен по адресу: http://localhost:8081

## Разворачивание в продакшн

Разворачивание сервиса в продакшн окружении с использованием Docker + Spring Boot Native Image:
1. `gradle :auth-service:bootBuildImage`
2. `docker run -d -p 8081:8081 --name auth-service hexaend/auth-service:latest`
3. Сервис будет доступен по адресу: http://localhost:8081
